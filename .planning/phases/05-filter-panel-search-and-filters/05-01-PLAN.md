In `wptools_imageconv_render_page()`, add filter panel HTML between the opening `wptools-imageconv-app` div and the existing `wptools-imageconv-toolbar` div.

Structure to add (insert BEFORE the toolbar div):

```html
<div id="wptools-imageconv-filter-panel" class="wptools-imageconv-filter-panel">
  <div class="wptools-imageconv-filter-row">
    <div class="wptools-imageconv-filter-group">
      <label for="wptools-imageconv-search">Search</label>
      <input
        type="text"
        id="wptools-imageconv-search"
        class="wptools-imageconv-filter-input"
        placeholder="Filename or title"
      />
    </div>
    <div class="wptools-imageconv-filter-group">
      <label for="wptools-imageconv-type">Format</label>
      <select id="wptools-imageconv-type" class="wptools-imageconv-filter-select">
        <option value="">All Formats</option>
        <option value="jpg">JPG</option>
        <option value="png">PNG</option>
        <option value="webp">WebP</option>
      </select>
    </div>
    <div class="wptools-imageconv-filter-group">
      <label for="wptools-imageconv-year">Year</label>
      <select id="wptools-imageconv-year" class="wptools-imageconv-filter-select">
        <option value="">All Years</option>
        <?php
        global $wpdb;
        $years = $wpdb->get_col(
          "SELECT DISTINCT YEAR(post_date) FROM {$wpdb->posts}
           WHERE post_type = 'attachment'
             AND post_status = 'inherit'
             AND post_mime_type IN ('image/jpeg','image/png','image/webp')
           ORDER BY post_date DESC"
        );
        foreach ($years as $y) {
          echo '<option value="' . esc_attr($y) . '">' . esc_html($y) . '</option>';
        }
        ?>
      </select>
    </div>
    <div class="wptools-imageconv-filter-group">
      <label for="wptools-imageconv-month">Month</label>
      <select id="wptools-imageconv-month" class="wptools-imageconv-filter-select">
        <option value="">All Months</option>
        <?php
        $month_names = [
          1 => 'January', 2 => 'February', 3 => 'March', 4 => 'April',
          5 => 'May', 6 => 'June', 7 => 'July', 8 => 'August',
          9 => 'September', 10 => 'October', 11 => 'November', 12 => 'December',
        ];
        foreach ($month_names as $num => $name) {
          echo '<option value="' . esc_attr($num) . '">' . esc_html($name) . '</option>';
        }
        ?>
      </select>
    </div>
  </div>
</div>
```

```html
</button>
```

Use K&R brace style and 2-space indentation throughout. All attribute values in double quotes. Wrap all string output with `esc_html__()` per existing patterns. The global `$wpdb` query is safe — no user input, all values hardcoded.
