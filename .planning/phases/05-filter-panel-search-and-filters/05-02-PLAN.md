Modify `imageconv_fetch_images` to accept an optional `params` object and merge it into the AJAX POST data. Then add the toggle and filter change event handlers.

**Step 1 — Refactor imageconv_fetch_images signature:**

Change the function signature from `function imageconv_fetch_images()` to `function imageconv_fetch_images(params)`. The params object is optional; default to empty object when not supplied.

Inside the function, replace the current `$.post(ajaxUrl, { action: ..., nonce: ... })` call with one that merges params:

```js
var postData = $.extend({
  action: 'wptools_imageconv_get_images',
  nonce:  nonce,
  page:   1,
}, params || {});

$.post(ajaxUrl, postData)
```

The rest of the `.done` and `.fail` callbacks remain unchanged.

**Step 2 — Add imageconv_get_filter_params helper (just before the Boot comment):**

```js
function imageconv_get_filter_params() {
  return {
    search: $('#wptools-imageconv-search').val() || '',
    type:   $('#wptools-imageconv-type').val()   || '',
    year:   parseInt($('#wptools-imageconv-year').val(),  10) || 0,
    month:  parseInt($('#wptools-imageconv-month').val(), 10) || 0,
  };
}
```

**Step 3 — Add toggle button handler (just before the Boot comment):**

```js
$('#wptools-imageconv-filter-toggle').on('click', function () {
  $('#wptools-imageconv-filter-panel').toggleClass('wptools-imageconv-filter-panel-open');
  $(this).toggleClass('is-active');
});
```

**Step 4 — Add debounced search handler (just before the Boot comment):**

```js
var imageconv_search_timer = null;

$('#wptools-imageconv-search').on('input', function () {
  clearTimeout(imageconv_search_timer);
  imageconv_search_timer = setTimeout(function () {
    imageconv_fetch_images(imageconv_get_filter_params());
  }, 400);
});
```

Use `input` event (not `keyup`) so paste and browser autofill are captured.

**Step 5 — Add select change handlers (just before the Boot comment):**

```js
$('#wptools-imageconv-type, #wptools-imageconv-year, #wptools-imageconv-month').on('change', function () {
  imageconv_fetch_images(imageconv_get_filter_params());
});
```

**Step 6 — Update Boot call:**

Change the boot call from `imageconv_fetch_images()` to `imageconv_fetch_images(imageconv_get_filter_params())` so initial load reads the current select state (all empty = no filters, consistent behaviour).
